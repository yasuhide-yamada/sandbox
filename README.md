# sandbox
## Usage

Maven
```xml
<repositories>
  <repository>
      <id>sandbox</id>
      <url>https://raw.github.com/yasuhide-yamada/sandbox/mvn-repo</url>
   </repository>
</repositories>

<dependencies>
   <dependency>
      <groupId>org.sandbox</groupId>
      <artifactId>sandbox</artifactId>
      <version>1.0</version>
   </dependency>
</dependencies>

```
